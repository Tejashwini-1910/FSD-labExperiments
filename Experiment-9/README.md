# Experiment 9

## Develop an AngularJS Form to Apply CSS and Events

---

## Aim

The aim of this experiment is to develop a form using **AngularJS** that applies **CSS styles dynamically** and handles **events such as button clicks, input changes, and form submission**.

---

## Technologies Used

- HTML5
- CSS3
- AngularJS
- JavaScript

---

## AngularJS Concepts Used

- **ng-app** – Initializes the AngularJS application  
- **ng-controller** – Defines the controller for the application  
- **ng-model** – Enables two-way data binding between input fields and the model  
- **ng-submit** – Handles form submission events  
- **ng-click** – Handles button click events  
- **ng-show** – Displays conditional messages  
- **ng-disabled** – Disables the submit button when the form is invalid
