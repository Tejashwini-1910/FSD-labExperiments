# How to Run React Experiments

Navigate to the project folder:

    cd experiment-folder-name

Install dependencies:

    npm install

Run the development server:

    npm start

Open in browser:

    http://localhost:3000

# Git Ignore

The following files should not be pushed to GitHub:

    node_modules
    build
    .env
    .DS_Store
    npm-debug.log
    yarn-error.log
